## Проект: Предсказание CTR для Advandex

## 📋 Описание проекта

Проект для рекламной платформы **Advandex**, работающей по модели аукциона. Задача — построить модель бинарной классификации, которая предсказывает вероятность клика (CTR) на рекламное объявление, и откалибровать её для точной оценки ставок.

### Почему это важно

- Если модель завышает CTR → платформа переплачивает за показы → убытки
- Если модель занижает CTR → платформа проигрывает аукционы → упущенная выгода
- Предсказанная вероятность 20% должна означать клик в среднем в 20 случаях из 100

### Данные

Датасет `ds_s16_ad_click_dataset` — аналитическая витрина показов рекламных баннеров. Каждая строка — одно событие показа.

**Ключевые группы признаков:**
- **Site:** категория сайта, домен, ID площадки
- **App:** категория, домен приложения
- **Device:** тип устройства, модель, соединение (Wi-Fi, 4G и т.д.)
- **Аукцион:** позиция баннера, анонимизированные признаки C1, C14-C21
- **ML-признаки:** машинно-сгенерированные фичи (ml_feature_1 — ml_feature_10)
- **Целевая переменная:** `click` (0 — нет клика, 1 — был клик)

> Данные анонимизированы: категории представлены хешами (например, `50e219e0`)

## спользуемые технологии

- **Python 3.9+**
- **Pandas, NumPy** — обработка данных
- **Scikit-learn** — LogisticRegression, SVM, GridSearchCV, метрики
- **Matplotlib, Seaborn** — визуализация, включая диаграммы калибровки
- **Joblib** — сохранение пайплайна

## Методология

### Этапы работы:

1. **Предобработка данных**
   - Заполнение пропусков (медиана / мода / категория `unknown`)
   - One-Hot Encoding для низкокардинальных признаков
   - Target Encoding для высококардинальных (например, `site_domain`)
   - Масштабирование числовых признаков

2. **Отбор признаков**
   - Фильтрационные методы: корреляция с целевой переменной
   - Методы-обёртки (Recursive Feature Elimination / SelectFromModel)

3. **Базовые модели**
   - LogisticRegression
   - SVM с линейным ядром
   - Подбор гиперпараметров через GridSearchCV

4. **Калибровка вероятностей**
   - Диаграммы калибровки (Calibration Curve)
   - Изотоническая регрессия (Isotonic Regression)
   - Оценка до/после калибровки

5. **Финальная оценка**
   - Комплексное сравнение моделей
   - Отчёт с графиками и выводами

##Ключевые метрики

| Метрика | Назначение |
|---------|------------|
| **PR-AUC** | Основная — качество на несбалансированных данных |
| **Log Loss** | Точность предсказанных вероятностей |
| **Brier Score** | Комплексная оценка точности и калибровки |
| **ECE / MCE** | Детальный анализ калибровки по диапазонам |
| **Precision, Recall, F1** | Вспомогательные метрики точности |
